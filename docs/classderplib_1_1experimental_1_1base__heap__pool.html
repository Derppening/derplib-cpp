Detailed Description

template<typename AllocStrategy, bool ThrowIfError = true>
class derplib::experimental::base_heap_pool< AllocStrategy, ThrowIfError >

A heap memory pool with support for different allocation strategies.

Template Parameters
AllocStrategyAn allocation strategy that is used to acquire/release memory in the pool.
ThrowIfErrorWhether to throw an exception if an error has occurred. Refer to the specific function for when the function will throw an error.
