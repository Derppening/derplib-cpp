template<typename T, std::size_t N>
class derplib::container::circular_queue< T, N >

A queue implementation using an std::array, implemented in a circular manner.

template<std::size_t SIZE>
derplib::container::circular_queue< T, N >::circular_queue ( const std::array< T, SIZE > &  cont)
explicitnoexcept

Conversion constructor from std::array.

Converts an std::array to a circular_queue. If the buffer of this object is smaller than the array, the first N elements will be copied. Otherwise, the elements from the array will be copied, and the remaining elements will be default-constructed.

The size() after initialization will be the equivalent of std::min(SIZE, N).
